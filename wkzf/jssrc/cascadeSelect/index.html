1.需要一个包含Select 元素的容器

2.依赖下发的数据格式，需满足 callBack({ "code": 200, "message": "成功", "result": { "cateGoryList": [{ "1": "专科机构", "childrens": [{ "20": "肛肠专科" }, { "21": "眼耳鼻喉" }, { "22": "儿科" }, { "23": "男科" }, { "24": "妇科" }] }, { "2": "美容养生", "childrens": [{ "8": "美容/SPA" }, { "9": "瘦身纤体" }, { "10": "产后塑形" }, { "17": "植发/养护" }, { "18": "足疗按摩" }, { "19": "男子美容" }] }, ] } });这样的格式。

3.必须设置级联的级数，默认为三级。如需要改动则可以设置 level 属性

4.可以通过classAttrs设置select的class.例如：classAttrs: ["prov", "city", "dist"]

5.对于表单提交的情况，需要设置select的name属性。同样可以通过设置nameAttr设置。例如：nameAttrs: ["prov", "city", "dist"]

6.可以通过defaultVals设置select的默认值.例如：defaultVals: ["北京市", "北京市", "东城区"]
